Q Format Notation: Qm.n format uses m bits for integer part and n bits for fractional part. For signed numbers, 1 bit is used for sign. Total bits = m + n (signed) or m + n + 1 (with sign bit).

Example: Q1.7 (signed) uses 1 bit for integer, 7 bits for fraction (8 bits total). Range: -2.0 to ~1.992. Resolution: 1/128 = 0.0078125
