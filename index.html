Verilog Tools

A collection of tools for Verilog / fixed-point development

Fixed-Point Converter

Convert between fixed-point binary/hex and floating-point decimal values. Configurable Q format with signed/unsigned support.

PE Calculator (Hex)

Compute butterfly and twiddle-factor rotation outputs for an FFT Processing Element using hex Q1.7 inputs. Includes full timing diagram.

PE Calculator (Real)

Same PE butterfly and rotation math using real (decimal) numbers. Useful for verifying expected results before fixed-point conversion.

IEEE 754 Float Converter

Convert between decimal and IEEE 754 hex/binary for half (16-bit), single (32-bit), and double (64-bit) precision. Shows sign, exponent, and mantissa breakdown.

Fixed-Point Calculator

Add, subtract, multiply, and divide fixed-point numbers with configurable Q format. Shows step-by-step arithmetic with overflow detection.
